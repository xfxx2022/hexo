# 软件介绍

PeaZip 是免费的文件存档器实用程序，基于 7-Zip，p7zip，Brotli，FreeArc，PAQ，Zstandard 和 PEA 项目的开源技术。

跨平台，功能全面但易于使用的 WinRar，WinZip 和类似的通用归档管理器应用程序的替代品，打开并提取200多种归档格式：001、7Z，ACE（\*），ARC，ARJ，BR，BZ2，CAB ，DMG，GZ，ISO，LHA，PAQ，PEA，RAR（\*\*），TAR，UDF，WIM，XZ，ZIP，ZIPX，ZST-查看支持的存档文件格式的完整列表，以 进行存档和提取。

该软件是根据开源 LGPLv3 许可证发行的，可免费使用（私有和专业），所有 PeaZip 软件包均为安全下载，不包含广告或有害软件。

# 功能特点

* **压缩格式转换**：直接将某个格式的文件转换成另一种压缩格式，无需手动先解压再压缩
* **脚本支持：**对于高级用户，你甚至还能调用命令行和编写脚本来做一些批量处理
* **查找重复文件：**在压缩包里帮你搜索出那些重复的文件
* **密码管理器：**Bandizip 收费的密码管理器功能在 PeaZIP 里是免费的
* **跨平台：**支持 Windows 和 Linux 系统
* **绿色便携：**官方提供绿色免安装版本
